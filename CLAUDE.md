# Locully Website — CLAUDE.md

## What this project is
The marketing website for **Locully** — an SEO & AI visibility agency based in Bangkok, Thailand.
- Live at: **locully.org**
- Hosted on: **Cloudflare Pages** (deployed via GitHub)
- Stack: React 18 + Vite + React Router v7 + Tailwind CSS + Radix UI + Framer Motion

## Routing

SPA with React Router. All routes defined in `src/App.jsx`.

**Cloudflare Pages SPA routing:** Handled natively via `public/_redirects` (`/* /index.html 200`). No server-side config needed.

Trailing-slash redirects are explicit in `App.jsx` for all `/ai-optimization/` and `/blog/` routes.

## Client deliverables workflow

### GEO Audit Reports
1. Run `geo-ai-visibility` subagent on client URL
2. Generate HTML report → save to `/Users/sunny/Projects/locully/audits/audit-[client-slug].html`
3. Export to PDF with Puppeteer (`export-pdf.cjs`) → `/Users/sunny/Projects/locully/audits/audit-[client-slug].pdf`
4. Open with `open -a Preview /Users/sunny/Projects/locully/audits/audit-[slug].pdf`

**Puppeteer gotchas:**
- Project uses `"type": "module"` — Puppeteer scripts must use `.cjs` extension
- Google Fonts need `setTimeout(4000)` after `networkidle0` to render properly
- `preferCSSPageSize: true` must be set in `page.pdf()` options
- PDF: `@page { size: A4; margin: 0 }` + `div.page { width: 210mm; height: 297mm }`

### SEO/AIO Proposals
- Template: `/Users/sunny/Projects/locully/proposals/proposal-seo-aio-template.html`
- Save completed proposals to `/Users/sunny/Projects/locully/proposals/`
- Trigger phrase: "Create an SEO/AIO proposal for [Client Name]"
- Always ask: domain, existing GEO audit, keyword CSV path, agreed pricing, ad spend, client brand font

## Key rules

- Never number blog posts in roadmap Mo. 3–4 or Mo. 5+
- Mo. 5+ in proposals is ONGOING — never imply it ends at month 6
- Keyword volumes: always use ranges (10,000–100,000), never single numbers
- GKP keyword data must come from real CSV exports — never fabricate
- Always verify client location before writing keyword targets — don't infer from website copy
- AIO is bundled under SEO — not separately priced in proposals
